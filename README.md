# Portfólio Individual (HJE)

Portfólio web com Front-end + Back-end em Django e persistência em PostgreSQL (Neon).

## Stack
- Django
- PostgreSQL (Neon)
- HTML/CSS (templates)

## Funcionalidades
- Exibe dados dinamicamente do banco (Profile, Projects etc.)
- Página de detalhe do projeto
- Formulário de contato salvando em PostgreSQL (ContactMessage)
- Admin para cadastrar conteúdos

## Rodar localmente
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
